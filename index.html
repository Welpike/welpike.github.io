Generate free redirect urls more easily with welpike.github.io !

welpike.github.io is a website that permits you to make long redirect urls (the strict opposit of bit.ly).

Generate now >>

Example

-> example.com

welpike.github.io redirect url : https://welpike.github.io/url.html?redirect=http%3A%2F%2Fexample.com%2F

Another example : https://welpike.github.io/url.html?redirect=https%3A%2F%2Fyoutu.be%2FdQw4w9WgXcQ&delay=2

Import the welpike.github.io JavaScript file !

For help you to redirect users, welpike.github.io has made a JavaScript file that you can import in your source code.
